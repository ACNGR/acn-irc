Greek Internet Relay Chat Network

Το ACN σας δίνει τη δυνατότητα να επικοινωνείτε κάνοντας chat γρήγορα, εύκολα και με ασφαλή τρόπο.

Συζητείστε για ό,τι σας απασχολεί, ανταλλάξτε απόψεις, κάντε νέες γνωριμίες και φίλους.

Αν έχετε δική σας ιστοσελίδα/blog και θέλετε να δημιουργήσετε Webchat για τους χρήστες σας, μπορείτε να επικοινωνήσετε μαζί μας στη διεύθυνση support@acn.gr. Οι υπηρεσίες μας παρέχονται εντελώς ΔΩΡΕΑΝ.

Για να συνδεθείτε στο Chat μέσω Browser χωρίς να εγκαταστήσετε κάποιο πρόγραμμα, πατήστε ΕΔΩ.

Μπορείτε να συνδεθείτε με IRC Client πληκτρολογώντας την εντολή: /server irc.hybridirc.com
